BASIC TAG

  • HTML headings are defined with the <h1> to <h6> tags.
    • <h1><\h1>
  • The <br> tag defines a line break
  • <hr> horizontal line:
  • The <p> element defines a paragraph.
    • <p><\p>

    lorem: create random word
    loremn: create n word
    lorem*n: create n line

    <b></b> : bold = <strong>

    <i></i> : italic = <em>

    <u></u> : underline

LIST TAG

<ul>: unordered list

    <ul>
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
    </ul>

<ol>: order list

write <ol type=" ">

Type Description
type="1" The list items will be numbered with numbers (default)
type="A" The list items will be numbered with uppercase letters
type="a"
type="I"
type="i" 
    type="1">;
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
    </ol>
